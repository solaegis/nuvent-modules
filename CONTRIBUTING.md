# Contributing to Nuvent Modules

Thanks for your interest in contributing!

## Adding a Module

1. Create the module directory under the appropriate provider (e.g., `gcp/`, `github/`)
2. Add a `module.nv` with the standard schema header
3. Run `task lint` to validate
4. Open a PR

## Guidelines

- All `module.nv` files must include the schema header
- Follow existing naming conventions
- Run `task lint` before submitting

By submitting a PR, you agree to the [CLA](CLA.md).
